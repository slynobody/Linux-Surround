# Linux / Steam-Deck: virtual Surround
Dlby @tmos-Sound for Steam-Deck / Linux. (use headphones, virtual 7.1 surround)

> git clone https://github.com/slynobody/Linux-Surround
> 
> cd Linux-surround
>
> chmod +x ./*.sh
>
> ./install.sh

afterwards: 

put on headphones. 
choose new Surround-Sound-Device. 
enjoy.
